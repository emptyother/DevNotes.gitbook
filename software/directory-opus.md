# Directory Opus

### Default Find-As-You-Type keyboard shortcuts

| Key | Type |
| :--- | :--- |
| `>` | Command |
| `?` | DOS command |
| ````` | WSL command |
| Any text | Filter |
| `*` | Filter Bar |
| `#` | Range |
| `=` | Search |
| `:` | Select |
| `@` | Tabs |
